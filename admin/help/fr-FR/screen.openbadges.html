JomBadger

L'aide principale se trouve dans le wiki de JomBadger. Vous êtes invité à participer à sa rédaction ;-).

Cette page est une aide rapide.

Après avoir mis en place le composant et le plugin, la première chose à faire est de créer une catégorie de badge, puis un badge. Vous pouvez aussi compléter les paramètres. Avant de se lancer plus loin dans l'interface, il faut rassembler les différentes ressources.

Commencez par trouver une image pour le badge, réfléchissez aux critères d'obtention et déterminer un nom au badge.

Ecrivez ensuite un article dans Joomla! qui indique ces différents éléments. Vous aurez besoin du lien vers cet article.

Enfin, dans l'interface d'administration de JomBadger, vous pouvez créer votre premier badge.

Principe d'attribution du badge :

  • le joueur effectue les actions que vous demandez (à vous de coder ces actions). JomBadger proposera des solutions sans codage
  • s'il rempli les critères, son badge peut être généré. A ce moment votre code doit insérer l'adresse email et l'id du badge dans la table ob_validate
  • Le joueur est alors redirigé vers la page "Acquérir un badge". Elle ne crée le badge que si les critères ont été validés (cf point ci-dessus). Cette page génère le badge et le stocke en base de données, puis propose d'envoyer le badge vers le backpack
